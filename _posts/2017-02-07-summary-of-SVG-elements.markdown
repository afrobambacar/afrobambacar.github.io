```
<svg width="400" height="200">
	<g>
		<rect></rect>
		<circle></circle>
		<line></line>
	</g>
</svg>
```

* ```svg``` svg 시작 태그
* ```g``` 그룹 태그 
* ```rect``` 사각형
* ```circle``` 원
* ```line``` 라인

[SVG Group Element and D3.js]: https://www.dashingd3js.com/svg-group-element-and-d3js
